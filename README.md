# Smart Algorithm Comparator

A Flask web application that benchmarks sorting and searching algorithms in real-time to demonstrate the impact of time complexity on performance, useful for understanding AI/ML data pipeline efficiency.

---

## 🌐 Live Demo
https://ai-project-2-007z.onrender.com  

⚠️ Note: Free Render instance may sleep after ~15 minutes of inactivity. First load may take ~30–50 seconds.

---

## 🛠 Tech Stack
- Python
- Flask
- JavaScript
- HTML5
- CSS3
- Gunicorn (WSGI Server)

---

## 🚀 Features
- Compare Bubble Sort `O(n²)` vs Merge Sort `O(n log n)`
- Compare Linear Search `O(n)` vs Binary Search `O(log n)`
- Real-time execution time measurement (high precision)
- Dark-themed responsive UI
- REST API backend with input validation and error handling

---

## 💡 Objective
To visually demonstrate how algorithm choice impacts performance and scalability in computing and AI/ML workflows.
